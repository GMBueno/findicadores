# Findicadores

Disponível em: [findicadores.com.br](https://findicadores.com.br)

## Sobre o site

Busque uma ação e acesse rapidamente:

- indicadores
- resultado
- cotação
- gráficos de cada um desses

## Rodar local

### Instalar

`npm install`

### Rodar

`npm run dev`

### Deployar

`npm run build`
`npm run deploy` // script customizado do pacote gh-pages
